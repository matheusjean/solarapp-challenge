## Como utilizar

Primeiro, instale os pacotes :

```bash
npm install
# or
yarn
```

Segundo, rode o projeto utilizando
```bash
npm run dev
# or
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) 

## Adicione a Sua key do Google Maps no .ENV
